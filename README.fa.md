# فروشگاه ساعت

یک برنامه مدرن فروشگاه آنلاین ساعت ساخته شده با Flutter و مدیریت حالت BLoC و طراحی پاسخگو.

## ویژگی‌ها

- **احراز هویت**: تأیید شماره تلفن مبتنی بر کد OTP و ثبت نام کاربر
- **کاتالوگ محصولات**: مرور ساعت‌ها بر اساس دسته‌بندی‌ها (کلاسیک، هوشمند، دیجیتال، رومیزی)
- **سبد خرید**: افزودن محصولات به سبد خرید و مدیریت تعداد
- **پروفایل کاربر**: مدیریت اطلاعات شخصی و آدرس‌ها
- **پیگیری سفارش**: نظارت بر وضعیت و تاریخچه سفارش‌ها
- **طراحی پاسخگو**: بهینه‌سازی برای اندازه‌های مختلف صفحه نمایش با استفاده از ScreenUtil
- **پشتیبانی از زبان فارسی**: پشتیبانی کامل RTL برای کاربران فارسی‌زبان

## پشته فناوری

- **فریمورک**: Flutter
- **مدیریت حالت**: الگوی BLoC
- **کامپوننت‌های UI**: ویجت‌های سفارشی با Material Design
- **فونت‌ها**: یکپارچه‌سازی Google Fonts
- **آیکون‌ها/تصاویر**: پشتیبانی Flutter SVG
- **پاسخگو**: ScreenUtil برای چیدمان‌های تطبیقی
- **معماری**: معماری پاک با جداسازی نگرانی‌ها

## شروع کار

### پیش‌نیازها

- Flutter SDK (^3.9.2)
- Dart SDK
- Android Studio / VS Code با افزونه‌های Flutter

### نصب

1. کلون کردن مخزن:
```bash
git clone <repository-url>
cd watch_store
```

2. نصب وابستگی‌ها:
```bash
flutter pub get
```

3. تولید فایل‌های مورد نیاز:
```bash
flutter pub run build_runner build
flutter pub run flutter_gen_runner
```

4. اجرای برنامه:
```bash
flutter run
```

### دستورات ساخت

- **ساخت دیباگ**: `flutter run`
- **ساخت انتشار**: `flutter build apk` یا `flutter build ios`
- **ساخت وب**: `flutter build web`

## ساختار پروژه

```
lib/
├── components/          # کامپوننت‌های قابل استفاده مجدد UI و تم‌ها
├── gen/                # فایل‌های تولید شده
├── res/                # منابع (رنگ‌ها، رشته‌ها، ابعاد)
├── screens/            # صفحات برنامه (احراز هویت، محصول، پروفایل)
├── widgets/            # ویجت‌های سفارشی
└── main.dart           # نقطه ورود برنامه
```

## وابستگی‌های کلیدی

- `flutter_screenutil`: ابزارهای طراحی پاسخگو
- `google_fonts`: تایپوگرافی سفارشی
- `flutter_svg`: پشتیبانی از تصاویر SVG
- `flutter_bloc`: مدیریت حالت
